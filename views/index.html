üìö Technical stuff

API was created using the MVC (models, views and controllers) structure.
It interacts with MongoDB.
Server running on port 5000

All Environment variables are in a .env file.
In your case create a .env file and declared the following variables:

LOCALURI=mongodb://127.0.0.1 :27017/follow-up
JWT_USER_SECRET=Your-JWT-User-Secret
JWT_ADMIN_SECRET=Your-JWT-Admin-Secret
